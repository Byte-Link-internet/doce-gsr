# Ferramentas

Algumas ferramentas podem te ajudar na escrita da documentação. Seguem algumas delas.


## Agentes conversacionais de IA

Os agentes conversacionais de IA como [ChatGPT](https://chatgpt.com/), [Gemini](https://gemini.google.com/) e [Deepseek](https://chat.deepseek.com/) geram, por padrão, documentação no formato Markdown. Copie e cole[^1] com responsabilidade: avalie se a resposta é realmente a esperada pelos avaliadores, e verifique se as referências estão corretas. E lembre-se de remover os emojis.

[^1]: Geralmente há o ícone :material-content-copy:

## Visual Studio Code

O Visual Studio Code (VS Code) é um editor de código-fonte gratuito, multiplataforma e extensível desenvolvido pela Microsoft. Embora tenha sido criado para programação, ele também é uma excelente ferramenta para escrever documentação técnica, especialmente quando ela é escrita em Markdown, como é o caso da plataforma Zensical.

Para quem mantém documentação em um repositório Git, o VS Code oferece praticamente tudo o que é necessário em um único ambiente: edição de texto, visualização do Markdown, integração com Git, terminal, controle de versões e centenas de extensões.
